# Professional Scientific Calculator

A sleek, responsive, and feature rich web based calculator built using HTML, CSS, and JavaScript. This project includes both a Normal Calculator and a Scientific Calculator with advanced mathematical capabilities and a modern UI.

## Features

### Normal Mode
- Basic arithmetic operations (+, −, ×, ÷)
- Percentage calculations
- Sign toggle (+ or −)
- Decimal support
- Keyboard input support
- Real time display updates

### Scientific Mode
- Trigonometric functions: sin, cos, tan
- Inverse trigonometric functions: sin⁻¹, cos⁻¹, tan⁻¹
- Logarithmic functions: log, ln
- Powers and roots: x², xʸ, √, ∛
- Constants: π, e
- Factorial (n!)
- Combinatorics: nCr, nPr
- Complex number support including imaginary i
- Expression parser with live preview
- Scientific notation formatting

## UI Highlights
- Modern dark theme
- Smooth animations and transitions
- Responsive design with mobile support
- Mode toggle between Normal and Scientific
- Dynamic font scaling for large numbers
- Interactive button feedback

## Advanced Capabilities
- Custom built expression parser

Handles:
- Operator precedence
- Nested expressions using parentheses
- Scientific notation
- Complex numbers (real and imaginary)

Supports real time evaluation preview.



## Project Structure
Everything including HTML, CSS, and JavaScript is contained in a single file for simplicity.
